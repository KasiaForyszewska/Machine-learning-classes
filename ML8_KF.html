def PCA_plot(score, coeff, labels=None):
    x = score[:,0]
    y = score[:,1]
    n = coeff.shape[0]
    scalex = 1.0/(x.max() - x.min())
    scaley = 1.0/(y.max() - y.min())
    plt.scatter(x * scalex, y * scaley, c = Y)
    for i in range(n):
        plt.arrow(0, 0, coeff[i,0], coeff[i,1], color = 'r', alpha = 0.5)
        if labels is None:
            plt.text(coeff[i,0] * 1.15, coeff[i,1] * 1.15, "Var" + str(i+1), color = 'g', ha = 'center', va = 'center')
        else:
            plt.text(coeff[i,0] * 1.15, coeff[i,1] * 1.15, labels[i], color = 'g', ha = 'center', va = 'center')
            
plt.xlim(-1,1)
plt.ylim(-1,1)
plt.xlabel("PC{}".format(1))
plt.ylabel("PC{}".format(2))
plt.grid()

Zadanie 4

Independent Component Analysis (ICA) to nienadzorowany algorytm ML, który służy do znajdowania niezależnych grup kolumn w danych.

Jest on podobny do PCA, przy czym w PCA staramy się znaleźć liniowe kombinacje zmiennych jako głównych składowych, pomiędzy którymi nie ma zależności. W ICA szukamy natomiast liniowych kombinacji zmiennych jako niezależnych składowych. Nie ma tu korelcji pomiędzy dowolnymi dwoma składnikami.

Algorytm ICA stosuje równanie $X= MD$, gdzie:

$X$ - macierz niezależnych składników $M$ - matryca mieszania $D$ - oryginalne dane

Algorytm ten znajduje niezależne składowe mnożąc macierz miksującą przez oryginalne dane. Jego celem jest znalezienie takiej Matrycy mieszanki „M” pomnożonej przez dane „D”, aby wszystkie kolumny w „X” były od siebie niezależne.

Dlaczego dane wykresy te różnią się zakresem wartości?

Mieszaninę sygnałów możemy porównać do rozmowy kilku osób. Algorytm ICA dąży do wyodrębnienia pojedyńczych głosów. Poprzez powyższe działania nie jest on jednak w stanie odzyskać oryginalnych współcznynników stojących przed sygnałem. Stąd zmiana zakresu wartości.
