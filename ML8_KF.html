def PCA_plot(score, coeff, labels=None):
    x = score[:,0]
    y = score[:,1]
    n = coeff.shape[0]
    scalex = 1.0/(x.max() - x.min())
    scaley = 1.0/(y.max() - y.min())
    plt.scatter(x * scalex, y * scaley, c = Y)
    for i in range(n):
        plt.arrow(0, 0, coeff[i,0], coeff[i,1], color = 'r', alpha = 0.5)
        if labels is None:
            plt.text(coeff[i,0] * 1.15, coeff[i,1] * 1.15, "Var" + str(i+1), color = 'g', ha = 'center', va = 'center')
        else:
            plt.text(coeff[i,0] * 1.15, coeff[i,1] * 1.15, labels[i], color = 'g', ha = 'center', va = 'center')
            
plt.xlim(-1,1)
plt.ylim(-1,1)
plt.xlabel("PC{}".format(1))
plt.ylabel("PC{}".format(2))
plt.grid()

Zadanie 4

Independent Component Analysis (ICA) to nienadzorowany algorytm ML, ktry suy do znajdowania niezalenych grup kolumn w danych.

Jest on podobny do PCA, przy czym w PCA staramy si znale liniowe kombinacje zmiennych jako gwnych skadowych, pomidzy ktrymi nie ma zalenoci. W ICA szukamy natomiast liniowych kombinacji zmiennych jako niezalenych skadowych. Nie ma tu korelcji pomidzy dowolnymi dwoma skadnikami.

Algorytm ICA stosuje rwnanie $X= MD$, gdzie:

$X$ - macierz niezalenych skadnikw $M$ - matryca mieszania $D$ - oryginalne dane

Algorytm ten znajduje niezalene skadowe mnoc macierz miksujc przez oryginalne dane. Jego celem jest znalezienie takiej Matrycy mieszanki M pomnoonej przez dane D, aby wszystkie kolumny w X byy od siebie niezalene.

Dlaczego dane wykresy te rni si zakresem wartoci?

Mieszanin sygnaw moemy porwna do rozmowy kilku osb. Algorytm ICA dy do wyodrbnienia pojedyczych gosw. Poprzez powysze dziaania nie jest on jednak w stanie odzyska oryginalnych wspcznynnikw stojcych przed sygnaem. Std zmiana zakresu wartoci.
